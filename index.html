常见问题

pap.er 是免费的吗？未来会收费吗？

pap.er 自 2017 年发布以来，一直为用户提供免费的高质量壁纸体验。我们计划在未来推出功能更丰富的付费版本，敬请期待！

pap.er 支持哪些 macOS 版本？新旧版 pap.er 有区别吗？

新版 pap.er 5.x （Mac Appstore 版本）需要 macOS 11.0 及以上系统版本，所有壁纸均由编辑精心挑选。旧版 pap.er 3.x 仍然兼容 Mac OS X 10.14+，但壁纸未经人工筛选。

pap.er 支持 iOS, Android 以及 Windows 系统吗？

目前 pap.er 暂不支持 iOS、Android 和 Windows 系统，但我们已将其纳入未来的发展计划。我们将针对不同的设备平台进行重新设计，以确保始终如一地提供卓越的用户体验。

如何设置自动更换壁纸？

您可以在 pap.er 界面右上角的设置入口中启用自动更换壁纸功能，并设置更换频率。您可以选择每周、每天或每小时更换一次。

如何在壁纸库加入我自己的图片？

您只需将图片直接放入 pap.er 的存储文件夹，并在设置中勾选“导入本地壁纸”即可。

pap.er 的壁纸来源是什么？

我们的壁纸均来自全球优秀的摄影师和设计师的作品。每张壁纸的下载按钮下方都附有出处链接，以确保所有壁纸均已获得授权（包括自动授权）且质量上乘。

我可以向 pap.er 提交我自己的壁纸作品吗？

我们期待与摄影师、设计师和品牌方直接合作，并将改进未来的新版本以更好地支持作品提交。欢迎您发送邮件与我们联系。
